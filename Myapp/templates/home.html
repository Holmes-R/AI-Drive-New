Connect Care Hub

Project Overview

This project connects different hospitals across the same city, enabling them to communicate with each other. Hospitals can view the resources available at other hospitals and request resources from them when needed.

As you are Dr. Anonymous, you have access to the following functionalities:

  • View records of different hospitals in the city
  • Search for a particular hospital based on name
  • Add appointments for patients to specific hospitals
  • View available resources of other hospitals, including medical equipment, doctors, and more

Only Admins can add hospitals*

API Endpoints

Here are the available API endpoints for interacting with this project:

  • Get List of Hospitals
  • Add Appointments
  • Request a Resource from Other Hospitals
